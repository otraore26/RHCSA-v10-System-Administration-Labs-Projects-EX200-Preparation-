## 🧪 Home Lab – RHCSA / RHCE (RHEL 10 + Hyper-V + Ansible)

I am focused on strengthening my Linux skills and developing hands-on experience with Ansible automation through scenario-based practice to prepare thoroughly for the RHCSA/RHCE exams. As part of the initial setup, we created a Red Hat account to download RHEL 10 and deploy it using either Hyper-V Manager or VirtualBox. I chose Hyper-V Manager since I already use it for other projects. This repository documents my home lab environment and includes screenshots to showcase the setup and progress.

### 🖥️ Lab Environment
- Host: Windows 11
- Hypervisor: Hyper-V Manager
- OS: Red Hat Enterprise Linux 10
- Automation: Ansible
- Focus Areas:
  - User & group management
  - Storage and file systems
  - Networking & services
  - Security basics
  - Ansible playbooks & automation
