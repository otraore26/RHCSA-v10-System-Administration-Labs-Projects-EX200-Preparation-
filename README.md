# RHCSA v10 System Administration Labs & Projects (EX200 Preparation)

This project offers a collection of hands-on Linux system administration labs aligned with the RHCSA v10 (EX200) exam objectives. It features practical exercises covering user and group management, filesystems and storage, networking configuration, service and process management, basic security hardening, shell scripting, and system troubleshooting. These labs are designed to build real-world administrative skills and serve as a personal learning portfolio for Red Hat–based Linux environments such as RHEL, AlmaLinux, Rocky Linux, and CentOS Stream.

 # 🛠️ Technologies Used
- Red Hat Enterprise Linux (RHEL 10) – Primary operating system for system administration labs
- Hyper-V Manager – Virtualization platform for building and managing the lab environment
- AlmaLinux / Rocky Linux / CentOS Stream – Alternative RHEL-based distributions for practice and compatibility testing
- Bash Shell – Command-line interface and scripting for automation and administration tasks
- Ansible – Automation tool for configuration management and task orchestration (RHCE-focused practice)
- OpenSSH – Secure remote access and system management
- Git & GitHub – Version control and documentation of lab work and projects
